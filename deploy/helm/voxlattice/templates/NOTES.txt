The server certificate is signed by a CA created inside this namespace.
In-cluster clients must trust it. Export the bundle with:

  kubectl -n {{ .Release.Namespace }} get secret {{ include "voxlattice.tlsSecretName" . }} \
    -o jsonpath='{.data.ca\.crt}' | base64 -d > voxlattice-ca.crt

Pass it to the LiveKit plugin as root_certificates.
{{- end }}
